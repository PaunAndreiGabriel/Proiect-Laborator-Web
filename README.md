# Portofoliu Web — Andrei Gabriel Păun

Acesta este un proiect de portofoliu personal, realizat ca pagină web statică. Prezintă date despre profil, educație, competențe și o listă de proiecte preluată din GitHub.

## Despre proiect

Pagina afișează informații personale și încarcă automat proiectele publice din contul GitHub. Dacă API-ul nu răspunde sau sunt prea puține proiecte disponibile, aplicația folosește o listă de rezervă.

## Tehnologii folosite

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub REST API

## Rulare locală

Proiectul nu are dependențe speciale. Ai nevoie doar de un browser modern și, recomandat, de un server local.

Varianta cea mai simplă este să folosești extensia **Live Server** din VS Code:

1. Deschide proiectul în VS Code.
2. Instalează extensia **Live Server**, dacă nu o ai deja.
3. Click dreapta pe `index.html` și alege **Open with Live Server**.
4. Se va deschide automat pagina în browser.

Adresa va fi ceva de forma:

```text
http://127.0.0.1:5500/index.html
```

Poți deschide și direct fișierul `index.html`, dar varianta cu server local este mai sigură pentru cererile către GitHub API.

## Structura proiectului

- `index.html` - structura paginii
- `styles.css` - stilurile aplicației
- `app.js` - logica pentru profil și proiecte
- `CV_Paun_Andrei_Gabriel.pdf` - sursa pentru datele din CV
